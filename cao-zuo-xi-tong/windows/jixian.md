### 8、虚拟内存清除

| 项目 | 详情 |
| :--- | :--- |
| 说明 | 在关机时，清除虚拟内存文件 |
| 检测步骤 | 进入“控制面板-&gt;管理工具-&gt;本地安全设置”，进入“本地策略-&gt;安全选项”： 将“关机：清除虚拟内存页面文件”配置为“已启用”。  将“用可还原的加密来存储密码”，配置为“不启用”。 |
| 符合性判定 依据 |  关机：清除虚拟内存页面文件”配置为“已启用”。 |
| 影响性分析  | 加固操作对业务无影响 |
