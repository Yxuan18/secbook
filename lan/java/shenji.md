# 代码审计

### 一、基础

代码审计工作流程  
配置分析环境→熟悉业务流程→分析程序架构→工具自动化分析→人工审计结果→报告

JSP生命周期   
客户端请求→web服务器→JSP容器→JSP网页→JSP网页实现类（.java）→JSP网页实现类（.class）→HTML→客户端请求
